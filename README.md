# Student Performance Prediction using Decision Tree Regression

##  Overview
This project predicts student marks based on study hours, attendance, and assignments using a Decision Tree Regression model.

##  Technologies Used
- Python
- Pandas
- Scikit-learn

##  Workflow
1. Load dataset
2. Data preprocessing
3. Train-test split
4. Model training using Decision Tree Regressor
5. Prediction
6. Evaluation using MSE and R2 Score

##  Evaluation Metrics
- Mean Squared Error (MSE)
- R2 Score

##  Result
The model predicts student marks with a small error. However, due to the small dataset, performance may not generalize well to real-world data.
